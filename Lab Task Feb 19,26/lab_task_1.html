HTML stands for HyperText Markup Language. It is used to create the structure of web pages. In this lab task, we are learning how to build a basic HTML5 document using proper tags and structure.

HTML also provides different text formatting elements to improve readability. For example, we can highlight text using mark tag, make text smaller using small tag, and break lines easily.
